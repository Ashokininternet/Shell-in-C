# Shell-in-C

A basic Unix shell implementation written in C. This project demonstrates how a command-line interface handles user input, process creation, and execution using system calls.

## Features
* **Command Execution:** Executes external programs by forking child processes.
* **Built-in Commands:** * `cd` - Change the current working directory.
  * `help` - Display information about the shell.
  * `exit` - Terminate the shell session.
  * `ls` - Lists the files/folders in the current working directory.
  * `rm` - Removes the selected file/folder .


## Prerequisites
* A C compiler (e.g., `gcc`)
* A Linux or Unix-like environment

## Building the Project
To compile the shell using the provided Makefile:
```bash
make
```
